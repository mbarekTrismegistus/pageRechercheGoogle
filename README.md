# Clone de la Page d'Accueil Google (TP)

Ce projet est une reproduction fidèle (mimétisme) de la célèbre page d'accueil du moteur de recherche Google. Il a été réalisé dans le cadre d'un Travail Pratique d'intégration web pour s'exercer à la structuration HTML et au stylisme CSS.

## Description du Projet

L'objectif de ce TP était de recréer l'interface utilisateur de la page de recherche Google en respectant l'agencement des éléments, les proportions et le design global. 

La structure de la page comprend :
- **Une barre de navigation (Header) :** Contenant les liens vers "Gmail", "Photos", le menu des applications Google (icône SVG) et une image de profil.
- **Le corps principal (Main) :** Centré avec le logo officiel de Google.
- **La barre de recherche :** Stylisée avec l'icône de loupe à gauche, et les icônes de recherche vocale (micro) et de recherche par image à droite.
- **Un bouton de raccourci :** Le bouton "Add shortcut" (Ajouter un raccourci) placé sous la barre de recherche.

## Technologies Utilisées

* **HTML5** : Pour la sémantique et la structure de la page.
* **CSS3** : Pour la mise en page (Flexbox/Grid), les couleurs, les marges et le design global (via le fichier `styles.css`).
* **SVG** : Utilisation d'icônes vectorielles en ligne et d'une image encodée en Base64 pour optimiser le chargement.
